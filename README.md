# samlbootstrap
This simplesamlphp module implements a bootstrap theme

# Installation

1. Modify simplesamlphp config file like so:
  
  'theme.use' => 'samlbootstrap:samlbootstrap',

2. Place the 'samlbootstrap' folder inside the 'modules' folder.
